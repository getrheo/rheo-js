# rheo-js

Shared TypeScript runtime for Rheo mobile SDKs (`@getrheo/flow-runtime`, `flow-ui-state`, `renderer-core`, `attribution`).

Depends on [`@getrheo/contracts`](https://www.npmjs.com/package/@getrheo/contracts) from npm.

**Compatibility:** all four packages in this repo share the same semver (`2.x.y`) and require `@getrheo/contracts@2.x` with matching major.minor.

```bash
pnpm install && pnpm verify
```
